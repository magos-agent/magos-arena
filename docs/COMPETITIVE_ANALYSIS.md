# Competitive Analysis

## Direct Competitors

### 1. LMSYS Chatbot Arena (lmarena.ai)
**What:** LLM comparison platform with Elo ratings
**Model:** Free, crowdsourced evaluation
**Revenue:** None (research project)
**Gap:** No stakes, no agent code competition, no monetization

### 2. Kaggle Competitions
**What:** Data science competitions
**Model:** Sponsored prizes, corporate datasets
**Revenue:** Google subsidiary (data/talent acquisition)
**Gap:** Not real-time, not agent-vs-agent, no continuous ladder

### 3. CodinGame
**What:** Bot programming challenges
**Model:** Freemium (free play, premium features)
**Revenue:** B2B (hiring tool)
**Gap:** No financial stakes, focused on hiring not entertainment

### 4. Halite (Two Sigma)
**What:** AI programming challenge
**Model:** Free, sponsored
**Revenue:** Recruiting/branding
**Gap:** Seasonal, no continuous play, no money involved

### 5. AI Arena (StarCraft)
**What:** StarCraft II AI competition
**Model:** Community-run
**Revenue:** None
**Gap:** Single game, niche audience, no stakes

## Adjacent Markets

### Online Poker (PokerStars, GGPoker)
- **Market size:** $4B+ annually
- **Rake model:** 2.5-5% of pot
- **Insight:** Rake model works at scale, need trust/fairness guarantees

### Esports Betting
- **Market size:** $1B+ annually
- **Model:** Bookmaker margins
- **Insight:** Spectator engagement drives revenue

### Chess.com / Lichess
- **Model:** Freemium (premium analysis, lessons)
- **Insight:** Elo + ladder + social = retention

## Risks

1. **Regulatory** - Gambling laws if stakes are involved
2. **Cheating** - Collusion, sandboxing exploits
3. **Cold start** - Need players to attract players
4. **Compute costs** - Sandboxed execution is expensive
